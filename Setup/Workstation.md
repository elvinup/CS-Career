## RTX

Install `rtx` to install most tools

```bash
curl https://rtx.pub/install.sh | sh

echo "eval \"\$(/home/elvin/.local/share/rtx/bin/rtx activate bash)\"" >> ~/.bashrc # or .zshrc if you're using zsh
```
